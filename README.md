# Atoms
CSS atoms for X-Team branded sites

## Setup
```
npm install
```

## View Documentation

```
npm link

cd docs
npm install
npm link atoms

npm start
```

## Building files

- `npm run build`: build the js bundle as normal
- `npm run build -- -w`: build the js bundle in watch-mode
